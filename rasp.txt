Install wiringPi :
cd /tmp
wget https://project-downloads.drogon.net/wiringpi-latest.deb
sudo dpkg -i wiringpi-latest.deb

Vous devez rajouter dans le fichier /boot/config.txt les deux lignes suivantes :
dtoverlay=pi3-disable-bt
enable_uart=1
